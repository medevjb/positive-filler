# FillMate Extension

![FillMate Logo](icons/icon128.png)

**FillMate** is a professional Chrome extension designed to automatically fill web forms with randomized, positive, and meaningful data. It works across all types of input fields, including text, email, password, textareas, and dropdowns, making form testing and demo filling fast, easy, and hassle-free.

---

## Features

- Automatically fills all **text input fields** with randomized words or sentences.
- Generates **emails, passwords, and positive phrases** dynamically.
- Handles **select/dropdown menus**: selects the first valid option automatically.
- Works **page by page**, activated only when you click the extension.
- Fully compatible with most websites, forms, and input types.
- Lightweight, clean, and **easy to use**.

---

## Installation

1. Download or clone the repository to your local machine.
2. Open **Google Chrome**, navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle at top right).
4. Click **Load unpacked** and select the folder containing this extension.
5. The extension icon will appear in the toolbar.

---

## How to Use

1. Navigate to any web page with input fields or forms.
2. Click the **FillMate** extension icon in the Chrome toolbar.
3. Click the **"Fill Page"** button in the popup.
4. All input fields and dropdowns on that page will automatically be filled with positive/randomized data.
5. Reload or navigate to another page to use it there again.

---

## Configuration

Currently, FillMate does not require any setup or configuration. You can customize the **positive words array, emails, or password patterns** in `content.js` if needed.
